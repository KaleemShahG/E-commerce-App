# Simple E-Commerce Website

A clean and simple e-commerce website built using **HTML, CSS, and Vanilla JavaScript**.

##  Features
- White clean UI
- 8 product categories
- 24 products (4 per category)
- Category-wise product sections
- Search functionality
- Category filter
- Product cards with image, title, description & price
- Responsive layout
- Simple header & footer

##  Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

##  Project Structure
This project demonstrates DOM manipulation, dynamic rendering, and basic UI logic using JavaScript.
